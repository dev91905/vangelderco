# Project Memory

## Core
Warm sand/ivory light mode, warm black dark mode — inspired by anthropic.com. Never cold gray.
Source Serif 4 headings (accent only), Inter body. No CRT, no grain, no brackets.
Audience: $100M+ donors. Tone: institutional authority, not marketing.
Deck slides must respect the horizontal frame; avoid vertical stacked cards.

## Memories
- [Design tokens](mem://design/tokens) — Anthropic-inspired warm neutral palette, typography, saturation rules
- [Dark mode](mem://design/dark-mode) — Warm blacks with brown undertone, not cold gray
- [Accent opacity](mem://design/accent-opacity) — CSS alpha tokens for accent elements
- [No serif constraint](mem://constraint/no-serif) — Serif only as accent for specific elements
- [Horizontal deck layout](mem://constraint/deck-horizontal-frame) — Preserve horizontal frame in deck
